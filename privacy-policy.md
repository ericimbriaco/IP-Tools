# Privacy Policy

The person responsible for data processing in connection with this Chrome Extension is:


---

## 2. Purpose of the Extension

The Chrome Extension “Show My IP” is designed solely to display the user’s current public IP address.

---

## 3. Data Processing

In order to display the public IP address, the user’s IP address must be technically transmitted to an external IP service so that the current public IP address can be determined and displayed.

The developer of this extension does **not collect, store, or share any personal data**.
The extension does **not store any data locally or on external servers**.

---

## 4. Third-Party Services

This extension uses an external service to retrieve the public IP address. For technical reasons, your IP address may be transmitted to this service. This is done solely for the purpose of displaying your public IP address.

---

## 5. Data Storage

No personal data is stored.

---

## 6. Data Sharing

No data is shared with third parties by the developer.

---
